# competitive-intel

研究和分析競爭對手策略、廣告、訊息傳遞和市場定位，產生可執行的競爭洞見。

## 概述

全面的競爭對手分析工具，將原始競爭資料轉化為策略洞見：
- 競爭對手訊息和定位
- 廣告策略和創意模式
- 定價和包裝模式
- 社交媒體和內容策略
- 市場定位和差異化

## 工作流程

1. **定義範圍** — 確認目標競爭對手、產業、管道和時間範圍
2. **收集資料** — 從公開來源收集廣告、登陸頁面、定價、訊息
3. **分析模式** — 對每個競爭對手套用分析框架
4. **綜合洞見** — 跨競爭對手比較，浮現趨勢和差距
5. **報告** — 提交附有清晰建議的調查結果

## 資料來源

- **WebSearch + WebFetch** — 預設方法，隨時可用
- 競爭對手網站、登陸頁面、定價頁面
- 評測網站（G2、Capterra、TrustRadius）
- 公開聲明、新聞稿、部落格文章
- 職缺發布（揭示策略優先事項）

## 輸出選項

- **Markdown 報告**（預設）— 執行摘要、競爭對手描述、比較
- **XLSX 試算表** — 結構化資料，含多個工作表供分析
- **PPTX 簡報** — 用於利益相關者分享的投影片組

## 法律與道德

- 僅使用**公開可訪問**的資訊
- 勿逐字抄寫競爭對手的創意或文案
- 勿嘗試訪問受限/私人內容
- 尊重智慧財產權
- 清楚標記調查結果為研究，非已驗證的聲明
